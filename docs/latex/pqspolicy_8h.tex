Authorizes a command request against the policy assigned to a privilege class. 

Evaluates the command policy associated with the supplied privilege level and determines whether the requested command is permitted. The function resolves the policy record assigned to {\ttfamily privilege}, validates the supplied command string, extracts the command verb used for policy matching, and applies the configured policy mode.

The authorization result depends on the matched policy mode\+:

\begin{DoxyItemize}
\item {\ttfamily pqs\+\_\+policy\+\_\+mode\+\_\+none} denies all command execution. \item {\ttfamily pqs\+\_\+policy\+\_\+mode\+\_\+restricted} permits only commands present in the policy allow list. \item {\ttfamily pqs\+\_\+policy\+\_\+mode\+\_\+forced} permits execution through the configured forced command. \item {\ttfamily pqs\+\_\+policy\+\_\+mode\+\_\+raw} permits commands unless explicitly denied by the policy.\end{DoxyItemize}
User-\/supplied command strings are also subject to shell-\/safety validation. Commands containing shell-\/control metacharacters or unsafe shell syntax are rejected before execution authorization succeeds. This prevents a permitted command verb from being used to carry additional shell expressions that would bypass the policy decision.

If {\ttfamily matched} is not NULL, it receives the address of the policy record used for the decision when a policy is resolved. The returned pointer refers to storage owned by {\ttfamily store} and remains valid only while the store remains valid and unmodified.


\begin{DoxyParams}{Parameters}
{\em store} & \mbox{[}const\mbox{]} A pointer to the initialized policy store. \\
\hline
{\em privilege} & \mbox{[}enum\mbox{]} The user privilege class whose assigned policy is evaluated. \\
\hline
{\em command} & \mbox{[}const\mbox{]} A pointer to the NUL-\/terminated command string to authorize. \\
\hline
{\em matched} & \mbox{[}out\mbox{]} An optional pointer that receives the matched policy record.\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
Returns true if the command is authorized by the matched policy; otherwise, returns false. 
\end{DoxyReturn}
