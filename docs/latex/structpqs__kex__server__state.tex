\doxysubsection{Detailed Description}
The PQS simplex server state structure. 

This structure holds the state information required for the server-\/side key exchange in the PQS protocol. It includes the server\textquotesingle{}s key identity, a session token hash, the server\textquotesingle{}s private and public keys for the asymmetric cipher, the server\textquotesingle{}s signature key, the server\textquotesingle{}s verification key, and the key expiration time.
