\doxysubsection{Detailed Description}
The PQS simplex client state structure. 

This structure holds the state information required for the client-\/side key exchange in the PQS protocol. It includes the client\textquotesingle{}s key identity, the remote server\textquotesingle{}s verification key, the client\textquotesingle{}s signature key, a session token hash, the client\textquotesingle{}s own verification key, and the expiration time of the keys.

Fields\+:
\begin{DoxyItemize}

\item {\ttfamily rverkey\+:} The remote asymmetric signature verification key (an array of PQS\+\_\+\+ASYMMETRIC\+\_\+\+VERIFY\+\_\+\+KEY\+\_\+\+SIZE bytes).
\item {\ttfamily sigkey\+:} The asymmetric signature signing key (an array of PQS\+\_\+\+ASYMMETRIC\+\_\+\+SIGNING\+\_\+\+KEY\+\_\+\+SIZE bytes).
\item {\ttfamily schash\+:} The session token hash (an array of PQS\+\_\+\+SCHASH\+\_\+\+SIZE bytes).

\item {\ttfamily expiration\+:} The expiration time, in seconds from epoch.
