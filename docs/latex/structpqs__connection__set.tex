\doxysubsection{Detailed Description}
Internal structure to manage the collection of connection states. 

This structure holds\+:
\begin{DoxyItemize}
\item an array of connection state structures ({\ttfamily conset}),
\item an array of boolean flags ({\ttfamily active}) that indicate whether each corresponding connection state is active,
\item the current number of connection states allocated ({\ttfamily length}),
\item and the maximum allowed number of connection states ({\ttfamily maximum}).
\end{DoxyItemize}

The structure is aligned for SIMD operations as required.
